BrawlNotes is a statical database system of BrawlHalla. BrawlHalla is an online multiplayer video game like the popular game Super Smash Bros. It is a free 2D platform fighting game where history’s greatest warriors brawl to prove who’s the best in an epic test of strength and skill. In order to analyze individual player and each game, keeping track of different player and game parameters is very important. That's when our system, BrawlNotes comes into play. BrawlNotes is an accessible and updatable repository of all the variables needed for analyzing BrawlHalla game. User can get all the recent information about the game and stay updated.
